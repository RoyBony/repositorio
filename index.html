Sobre mí

Soy un desarrollador web en formación con especial interés en el backend y el manejo de bases de datos. Tengo conocimientos básicos en HTML, CSS, JavaScript y en la creación de estructuras de bases de datos con MySQL y MongoDB. Me gusta construir soluciones que conecten el frontend con el backend para lograr aplicaciones dinámicas y funcionales. Mi objetivo es crecer como desarrollador backend, adquiriendo experiencia en la gestión de datos y seguridad de aplicaciones web, mientras contribuyo a proyectos innovadores en el área tecnológica.

Mis Habilidades

Estas son algunas de las áreas en las que tengo experiencia:

Frontend

Creación de interfaces web modernas usando HTML, CSS y JavaScript. Manejo de tecnologías de apoyo como Bootstrap, FontAwesome y SweetAlert.

Backend

Desarrollo backend con Python, Java y Node.js. Creación de APIs REST, autenticación básica y despliegue en entornos con contenedores.

Bases de Datos

Diseño y gestión en MySQL y PostgreSQL. Consultas SQL, creación de tablas, relaciones y análisis con JOINs.

GitHub/Git

Control de versiones con Git y gestión de proyectos en GitHub: ramas, commits, pull requests y resolución de conflictos.
